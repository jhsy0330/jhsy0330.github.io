安装Arch linux 这有一篇简洁的guide

前言

众所周知，Arch安装是一个极其不友好的一个过程。很多linux入门玩家选择的应该都是一些具有图形化安装界面的linux发行版，比如Ubuntu，Deepin，CentOs等这些相对来说对于安装过程不需要什么学习成本，只需要按照常规逻辑安装就可以装上了。但是对于没有图形化安装界面的Arch，对于初次安装的同学就有些不那么友好。可能某一步出错或者是忘记步骤出现问题就会导致系统无法启动或者安装后无法正常使用（基本都是引导或者网络连接的问题）。我初次安装的时候根绝Arch wiki的Installation guide也出错了好几次，多次尝试之后学会了如何安装arch，在此整理分享出比官方guide更简洁的安装教程

制作启动优盘

这一步我相信点开这篇文章的读者都会的一个操作，我就不赘述了

搜索并下载arch的iso文件用软碟通制作启动优盘

进入到Arch live CD环境

不同的BIOS有不同的开机启动菜单热键，大部分应该都是F12或者F8，如不确定查询主板手册或者网络搜索

在启动菜单选择U盘启动后进入到live CD环境

检查网络连接并连接网络

有线网络

有线网络默认是自动连接的，可以ping任何一个网站测试连接

ping archlinux.org

没有连接的话可能是没有自动分配ip地址，需要手动启动dhcp服务

dhcpcd

无线网络

无线网络使用iwctl连接wifi

我暂时还没用过wifi安装arch，暂时跳过这里。有需要可以直接查阅官方wiki

https://wiki.archlinux.org/index.php/Iwd#iwctl

磁盘分区并格式化

这里我使用fdisk对硬盘进行分区

磁盘分区

分区有传统的MBR和GPT分区表，这里我使用GPT。我相信现在大部分电脑都是UEFI+GPT引导方式启动了，所以就不再使用老旧的BIOS+MBR了

首先列出我们要分区的硬盘

fdisk -l

我们找到要进行分区的硬盘，选择对应的设备名

fdisk /dev/sda		#这里sda就是指的硬盘

首先要建立gpt分区表，进入fdisk后会选择操作，这里我们按下g选择建立gpt分区表，如果对操作不懂的可以按m键查看帮助

接下来按下n键，新建一个分区。命令符会提示让你选择分区号，起始位置，分区大小，只需要设置分区大小其他都默认或者根据你的需求设置。分区大小可以使用+512M或者+100G这种格式轻易的设置分区大小

对fdisk操作不熟悉的可以查询fdisk使用指南

操作完成之后记得按w键保存分区表

这里我分了三个区

  • EFI启动分区 512M
  • linux文件分区 200G
  • swap分区 20G

磁盘格式化

接下来我们就要格式化分区了

使用mkfs指令格式化分区 /sda就是刚刚操作的硬盘，紧跟着的数字就是分区号

mkfs.fat -F32 /dev/sda1
mkfs.ext4 /dev/sda2

然后要建立swap分区，并启用swap分区

mkswap /dev/sda3
swapon /dev/sda3

挂载分区

硬盘分区格式化之后还没完，我们还要接着把格式化好的分区挂载到文件系统上才能安装

使用mount挂载分区，因为efi分区需要挂载在系统目录下，所以还要再新建一个目录作为efi启动分区的挂载点

mount /dev/sda2 /mnt
mkdir /mnt/boot
mount /dev/sda1 /mnt/boot

我在某一次安装的时候没有挂载efi分区，导致我后来系统没办法启动

安装arch

使用pacstrap安装内核

pacstrap /mnt base linux linux-firmware

如果想修改镜像源的话，可以修改文件/etc/pacman.d/mirrorlist

系统配置

生成Fstab

使用genfstab生成tstab文件，fstab是系统记录挂载信息的文件

genfstab -U /mnt >> /mnt/etc/fstab

安装编码

编辑/mnt/etc/locale.gen 然后移除需要编码的注释符号

我们一般使用en_US.UTF-8和zh_CN.UTF-8，把他们前面的#删除掉就可以了，然后保存

设置hostname

创建/mnt/etc/hostname文件并直接键入主机名

例如我的主机叫jhsy就直接在hostname输入jhsy并保存

设置hosts

hosts文件位于/mnt/etc/hosts，我们用编辑器打开并编辑

# Static table lookup for hostnames.
# See hosts(5) for details.
127.0.0.1	jhsy
::1		localhost
127.0.0.1	jhsy.domain	jhsy

这里jhsy是我的主机的名字，这里只需要替换成你的主机的名字即可

本地化

使用arch-chroot进入我们刚刚安装好的系统

arch-chroot /mnt

输入这个指令后你的终端应该就会改变名字了，就说明进入了新的系统

生成编码

刚刚设置了编码，这个时候我们要在系统里生成编码

locale-gen

片刻之后就提醒我们编码已经安装完成了

设置LANG变量

直接使用localectl设置LANG变量

localectl set-locale LANG=en_US.UTF-8

这里你也可以改成zh_CN.UTF-8

设置时区

我们使用软连接直接在地区文件链接到localtime

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

设置时间

生成/etc/adjtime

hwclock --systohc

hwclock –systohc的作用是将系统时间写入硬件时间

安装引导程序

安装grub efibootmgr intel-ucode程序设置引导

pacman -S grub efibootmgr intel-ucode

在efi分区中创建一个grub的目录存放grub配置

生成grub.cfg文件和安装grub程序

mkdir /boot/grub
grub-mkconfig > /boot/grub/grub.cfg
grub-install --target=x86_64-efi --boot-directory=/boot

这里可能会出现安装失败的情况，可能是你的主板设置没有启动UEFI或者没有关闭CSM，CSM是一种同时使用BIOS和UEFI的兼容模式，只有仅使用UEFI模式启动这里才能安装成功grub，如果失败的话不用着急重新设置一下主板就可以了

设置密码

输入passwd然后会要求你输入新密码

安装基本程序

这里我想安装vim编辑器，联网还需要dhcpcd，还有基本的开发组件base-devel，其他的可以稍后再安装

pacman -S vim dhcpcd base-devel

如果创建新用户的话记得安装sudo，不然是没法使用sudo命令的
